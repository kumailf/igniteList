# IgniteList 🔥

一个简单而有趣的每日待办事项应用，完成待办事项时会显示夸张的动画效果和音效来鼓励你！

## 功能特点

- ✅ **简单易用**：只有一个页面，专注于你的待办事项
- 🎉 **夸张的动画效果**：完成待办事项时会有炫酷的庆祝动画
- 🔊 **鼓励音效**：完成任务时播放音效（需要添加音效文件）
- 💾 **数据持久化**：待办事项会自动保存，下次打开应用时会自动恢复
- 🔄 **每日自动重置**：每天24点过后，所有待办事项会自动重置为待完成状态，让你每天都有新的开始。无需重启应用，系统会自动检测日期变化
- 🏆 **连续完成天数**：每个待办事项都会记录连续完成天数，完成时会显示"N连胜！"的庆祝弹窗，激励你保持好习惯
- 📱 **跨平台支持**：同时支持 iOS 和 Android

## 开始使用

### 安装依赖

```bash
flutter pub get
```

### 添加音效（可选）

1. 将你的成功音效文件命名为 `success.mp3`
2. 放置到 `assets/sounds/` 目录中
3. 重新运行 `flutter pub get`

音效文件建议：
- 格式：MP3 或 WAV
- 时长：1-3 秒
- 风格：欢快、鼓励性的音效

免费音效资源：
- https://freesound.org/
- https://www.zapsplat.com/
- https://mixkit.co/free-sound-effects/

### 运行应用

```bash
# iOS
flutter run -d ios

# Android
flutter run -d android
```

## 使用说明

1. **添加待办事项**：在输入框中输入待办事项，点击添加按钮或按回车键
2. **完成待办事项**：点击待办事项左侧的圆圈来标记为完成
3. **删除待办事项**：点击右侧的删除按钮
4. **享受庆祝动画**：完成待办事项时会自动播放庆祝动画和音效，显示连续完成天数（如"3连胜！"）
5. **每日重置**：每天24点过后，所有待办事项会自动重置为待完成状态，让你每天都有新的开始
   - 应用启动时自动检查
   - 应用从后台恢复时自动检查
   - 应用在前台运行时，每分钟自动检查一次
   - 无需手动重启应用
6. **连续完成记录**：每个待办事项都会记录连续完成天数
   - 如果昨天完成过，今天完成时连续天数+1
   - 如果中断了，连续天数重新开始
   - 完成时会显示"N连胜！"的庆祝弹窗
