# IgniteList v1.0.1

## 🎉 首次发布

IgniteList 是一个充满活力的每日待办事项应用，完成任务时会有夸张的动画和音效来鼓励你！

## ✨ 主要功能

### 📝 待办事项管理
- ✅ 添加、完成、删除待办事项
- 🔄 拖拽排序：长按待办项任意位置即可拖动排序
- 📊 显示连续完成天数：每个事项后显示连续完成天数标签
- 📍 新事项默认添加到列表顶部

### 🎊 完成庆祝
- 🎉 完成待办事项时播放夸张的庆祝动画
- 🎵 支持多个语音包选择（aqua、mea、冬雪莲、松冈修造）
- 🔇 支持静音模式
- 💫 彩纸动画效果
- 👆 庆祝弹窗点击屏幕任意位置关闭

### 📅 每日重置
- 🔄 每日午夜自动重置所有已完成事项
- 📈 自动计算连续完成天数
- 💾 数据本地持久化保存

### 🎨 界面设计
- 💙 水蓝色主题
- 📱 现代化 Material Design 3 界面
- ✨ 流畅的动画效果

## 📦 安装

下载 `app-release.apk` 文件并安装到 Android 设备上。

**系统要求：**
- Android 5.0 (API 21) 或更高版本

## 🔧 技术栈

- Flutter 3.9.2+
- Dart SDK
- 本地存储：shared_preferences
- 音频播放：audioplayers
- 动画效果：confetti
